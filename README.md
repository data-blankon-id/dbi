# dbi


Add the weather API from npm

```
$ dbi add weather
```

Start the server

```
$ dbi start
```

Use pm2 

```
$ dbi start -d
```

Not satisfy with the 3000 as the default port?

```
$ dbi start -p 8000
```

Then open `http://localhost:3000`, you'll be redirected to a swagger page.
